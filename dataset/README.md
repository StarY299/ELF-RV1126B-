# 吸烟检测数据集

## 规模
- 总图片数：20,000+ 张
- 标注类别：4类（no_smoking / smoking / fire / smoke）
- 标注格式：YOLO 格式
- 数据来源：公开吸烟检测数据集 + 多场景实拍采集

## 数据划分
- 训练集：80%（约 16,000 张）
- 验证集：10%（约 2,000 张）
- 测试集：10%（约 2,000 张）

## 场景覆盖
室内（教室、走廊、大厅、楼梯间）、半室外（阳台、站台、建筑入口），
涵盖多种光照条件（强光、逆光、弱光）、不同视角、远距离小目标、
遮挡、烟雾稀薄等挑战性样本。

## 增强策略
随机裁剪、亮度抖动、镜像翻转、高斯噪声等数据增强。
